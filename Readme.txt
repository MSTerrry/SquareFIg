Программа рассчитывает площадь n-угольника через формулу площади гаусса.
На вход могут подаваться точки в разных форматах: примеры (1;1), 1.1;1.
В качестве разделителя целой и дробной части можно использовать точку или запятую,
Координаты x и y разделяются символом ';'. В один текстбокс идет одна точка.
При необходимости большего количества точек, их можно добавить через Add point, либо удалить
через Remove point в противоположном случае.
Кнопка Clear чистит все текстбоксы.
Calculate рассчитывает площадь.
В случае некорректного ввода программа выставит дефолтное значение 0 вместо координаты,
которую не смогло распарсить и выдаст сообщение об ошибке с некорректной координатой.
